Getting Started

Basic Usage

1. Create filters using the form in the Demo tab

2. Add filters to rule groups using AND/OR logic

3. Build and preview your query

4. Save or export the query for later use

API Examples

JSON Structure

{
  "query": {
    "type": "semantic",
    "vector": ["embedding", "array"],
    "groups": [
      {
        "logic": "and",
        "rules": [
          {
            "field": "category",
            "operator": "equals",
            "value": "feedback"
          }
        ]
      }
    ]
  }
}
Advanced Features

Nested Groups

You can nest groups inside other groups to create complex logic:

(A AND B) OR (C AND D)

Saved Filters

Filters are stored locally in your browser and can be reused across different queries.
