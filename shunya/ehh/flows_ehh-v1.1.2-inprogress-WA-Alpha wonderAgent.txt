Action × Entity × Role Snapshot
  - Roles: Admin, Analyst, Viewer
  - Action vocabulary: initialize, create, read, update, delete, list, validate, cache, monitor, audit, search, filter, sort, paginate, authorize, authenticate, publish, subscribe
  - Entities: portfolio, project, session, workflow, dataset, task, watchlist, agent, room, chat session

Example Matrix (for reference automation):
  - Admin | Portfolio | create (bootstraps entity, uses resolvePath, fires authorize + audit hooks)
  - Analyst | Project | read (enforces authorize + filter, logs via monitor)
  - Viewer | Session | list (reads via cache, applies paginate + sort)
  - Admin | Dataset | update (validates schema, triggers cache + audit)
  - Analyst | Task | search (uses search + filter + paginate)
  - Viewer | Watchlist | view (applies authorize, returns read-only payload)
